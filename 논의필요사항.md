## 2. 복수선택 채점 방식 [상태: 결정됨 — 2026-04-08 / 구현 완료]

`multiple_answers` 유형 문항 생성 시 교수자가 채점 방식을 직접 선택.

| **옵션** | **설명** | **기본값** |
|---|---|---|
| 전체 정답 시에만 만점 (`all_correct`) | 오답 하나라도 포함 시 0점 | 기본값 |
| 정답 비율 배점 (`partial`) | 정답 개수 / 전체 정답 수 × 배점 | 선택 |

- 문항 데이터에 `scoringMode` 필드 추가
- 감점 방식은 퀴즈 단위 옵션으로 결정됨 (결정됨 §8)

---

## 3. 지각 제출 감점 방식 [상태: 결정됨 — 2026-04-09 / XN 범위 외]

Canvas Gradebook에 원점수만 넘기는 구조로 확정됨에 따라, 감점 계산은 Gradebook(Canvas Late Policies)에서 처리. XN Quizzes는 지각 제출 허용 여부(`allowLateSubmit`) 및 허용 기간(`lateSubmitHours`)만 관리.

- Canvas 감점 설정 경로: Gradebook → 우측 상단 톱니바퀴 → Late Policies 탭

---

## 4. 문제은행 문항 삭제 시 처리 [상태: 결정됨]

문제은행-퀴즈 연동이 독립 복사 방식으로 확정됨에 따라, 문제은행 문항 삭제는 이미 퀴즈에 복사된 문항에 영향 없음.

---

## 5. 문제은행-퀴즈 연동 방식 [상태: 결정됨: 독립 복사]

문제은행에서 퀴즈로 문항 추가 시 해당 시점에 데이터를 복사. 이후 문제은행 수정은 퀴즈에 반영되지 않음.

## 8. 복수선택 감점 방식 [상태: 결정됨 — 2026-04-10 / 구현 완료]

`multiple_answers` 유형에서 오답 선택 시 감점 적용 여부를 **퀴즈 단위 옵션**으로 제공.

- 퀴즈 생성/편집 시 "채점 설정" 영역에서 감점 방식 ON/OFF 가능
- 감점 공식: `(정답 선택 수 - 오답 선택 수) / 전체 정답 수 × 배점` (최소 0점)
- `scoringMode === 'partial'`인 복수선택 문항에만 적용 (all_correct 모드에는 영향 없음)
- 퀴즈 데이터에 `penaltyEnabled` 필드 추가 (기본값: false)
- 연관 결정: §2 (복수선택 채점 방식)

---

## 9. 응시 중 문항 수정 범위 [상태: 결정됨 — 2026-04-10 / 구현 완료]

기존 퀴즈와 동일하게, 이미 제출한 학생이 있더라도 **문항 수정(텍스트, 정답, 선택지 등)을 허용**한다.

- 문항이 변경될 경우, 변경 시점(`questionsModifiedAt`) 기록
- 채점 대시보드에서 "수정 전 제출" 필터를 제공하여, 문항 변경 이전에 제출한 학생만 구분하여 확인 가능
- 교수자는 해당 학생들만 필터링한 뒤 재채점 또는 수동 점수 조정 가능
- Canvas Classic Quizzes 방식 준용 (수정은 자유, 기존 제출자에 대해서는 교수자가 수동으로 재채점 트리거)
