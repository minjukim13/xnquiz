## 2. 응시 중 문항 수정 범위 [상태: 미결]

### Canvas LMS 리서치 결과

**Canvas Classic Quizzes**:
- 퀴즈 발행 후에도 문항 편집 가능
- 응시 중 세션: 응시 시작 시점 스냅샷 기준 유지, 편집 내용 미반영
- 이미 제출된 응답: 편집만으로는 재채점 미발생, 교수자가 수동으로 Regrade 트리거해야 함
- 이후 응시자: 편집된 버전 즉시 노출

**Canvas New Quizzes (현재 표준)**:
- 첫 응시자 발생 후 편집 기능 일부 잠금
- 문항 텍스트 오탈자 수정: 허용
- 정답 변경, 선택지 추가/삭제, 문항 삭제, 배점 변경: 제한 또는 경고 후 재채점 필요

### 권고안

Canvas New Quizzes 접근 방식 채택을 권장.

| **수정 항목** | **권고** | **비고** |
|---|---|---|
| 문항 문구 오탈자 수정 | 허용 | 텍스트만, 의미 변경 없는 수정 |
| 선택지 텍스트 수정 | 허용 (주의) | 기존 응시자 스냅샷 기준 유지 |
| 정답 변경 | 허용 + 재채점 트리거 | 제출 완료 응답에 소급, 진행 중 세션 제외 |
| 배점 변경 | 차단 | 응시 발생 후 변경 불가 |
| 문항 삭제 | 차단 | 응시 기록 있으면 삭제 불가 |
| 문항 추가 | 차단 | 응시 중 추가 불가 |
| 선택지 추가/삭제 | 차단 | 응시 중 구조 변경 불가 |

**결정이 필요한 쟁점**: 정답 변경 + 소급 재채점 자동화 여부 (자동 vs 교수자 수동 트리거)

# 결정됨

## 1. 재채점 가능/불가 유형 [상태: 결정됨 — 2026-04-08]

| **재채점 가능** | **재채점 불가 (수동채점 유지)** |
|---|---|
| 객관식 단일/복수, 참/거짓, 단답형, 수치형, 연결형, 다중빈칸 | 서술형, 파일 업로드 |

- 수동 조정 문항은 재채점에서 제외
- 재채점 완료 알림: MVP 이후 과제

---

## 2. 복수선택 채점 방식 [상태: 결정됨 — 2026-04-08 / 구현 완료]

`multiple_answers` 유형 문항 생성 시 교수자가 채점 방식을 직접 선택.

| **옵션** | **설명** | **기본값** |
|---|---|---|
| 전체 정답 시에만 만점 (`all_correct`) | 오답 하나라도 포함 시 0점 | 기본값 |
| 정답 비율 배점 (`partial`) | 정답 개수 / 전체 정답 수 × 배점 | 선택 |

- 문항 데이터에 `scoringMode` 필드 추가
- 감점 방식은 미결로 유지 (미결 §1)

---

## 3. 재응시 후 기존 채점 기록 확인 [상태: 결정됨 — 2026-04-08 / 구현 완료]

재응시 후에도 기존 응시 회차별 채점 기록을 학생이 확인할 수 있어야 한다. 점수 정책(`scorePolicy`: 최고점/최신/평균)에 따른 최종 점수 계산은 유지.

- 학생 뷰 종료 카드: 복수 응시 이력이 있는 경우 "응시 기록 N회" 표시 및 회차별 점수 확인 가능

---

## 4. 복수 응시 점수 정책 소급 변경 [상태: 결정됨 — 2026-04-09 / XN 범위 외]

Canvas Gradebook에 원점수만 넘기는 구조로 확정됨에 따라, 점수 정책 변경 시 소급 적용은 Gradebook 영역. XN Quizzes는 응시 회차별 원점수만 저장하며 소급 재계산을 담당하지 않음.

---

## 5. 지각 제출 감점 방식 [상태: 결정됨 — 2026-04-09 / XN 범위 외]

Canvas Gradebook에 원점수만 넘기는 구조로 확정됨에 따라, 감점 계산은 Gradebook(Canvas Late Policies)에서 처리. XN Quizzes는 지각 제출 허용 여부(`allowLateSubmit`) 및 허용 기간(`lateSubmitHours`)만 관리.

- Canvas 감점 설정 경로: Gradebook → 우측 상단 톱니바퀴 → Late Policies 탭

---

## 6. 문제은행 문항 삭제 시 처리 [상태: 결정됨]

문제은행-퀴즈 연동이 독립 복사 방식으로 확정됨에 따라, 문제은행 문항 삭제는 이미 퀴즈에 복사된 문항에 영향 없음.

---

## 7. 문제은행-퀴즈 연동 방식 [상태: 결정됨: 독립 복사]

문제은행에서 퀴즈로 문항 추가 시 해당 시점에 데이터를 복사. 이후 문제은행 수정은 퀴즈에 반영되지 않음.

---

## 8. 엑셀 문항 업로드 지원 유형 [상태: 결정됨]

행 단위 구조로 표현 가능한 유형만 지원.

| **유형** | **코드** | **지원 여부** |
|---|---|---|
| 객관식 | multiple_choice | 지원 |
| 참/거짓 | true_false | 지원 |
| 단답형 | short_answer | 지원 |
| 서술형 | essay | 지원 (문항 텍스트만) |
| 수치형 이하 나머지 | — | 미지원 |

---

## 9. 엑셀 일괄 채점 업로드 정책 [상태: 결정됨]

| **항목** | **결정 내용** |
|---|---|
| 학번 미매칭 처리 | 1건이라도 미매칭 시 전체 업로드 취소 |

| 오류 발생 시 | 부분 저장 없음. 전체 취소 후 재업로드 유도 |
