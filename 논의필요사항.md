## 2. 복수선택 채점 방식 [상태: 결정됨 / 전역 설정으로 통합]

`multiple_answers` 유형의 채점 방식과 감점 정책을 **전역 설정(조직 단위)**으로 통합 관리.

- **퀴즈 관리 페이지 상단 톱니바퀴 아이콘 → "퀴즈 전역 설정" 다이얼로그**에서 설정
- 모든 퀴즈에 일괄 적용되는 조직 단위 정책 (대학별로 다를 수 있는 요구사항을 옵션화)
- 개별 퀴즈/문항 단위 설정(`penaltyEnabled`, `scoringMode`) 제거 → 전역으로 일원화

**채점 방식 (multipleAnswersScoringMode)**

| **옵션** | **설명** | **기본값** |
|---|---|---|
| 전체 정답 시에만 만점 (`all_correct`) | 모든 정답 선택 + 오답 미포함 시 만점, 그 외 0점 | 기본값 |
| 정답 비율 배점 (`partial`) | 정답 선택 비율에 따라 부분 점수 부여 | 선택 |

**감점 방식 (penaltyMethod, partial 선택 시만)**

| **옵션** | **공식** | **특징** |
|---|---|---|
| 감점 없음 (`none`) | 정답 수 / 전체 정답 수 x 배점 | 오답 선택 시 감점 없음 |
| 오답 차감 (`right_minus_wrong`) | (정답 수 - 오답 수) / 전체 정답 수 x 배점 (최소 0점) | 오답 1개당 정답 1개분 차감 |
| 추측 보정 감점 (`formula_scoring`) | (정답 수 - 오답 수/(선택지 수-1)) / 전체 정답 수 x 배점 (최소 0점) | 찍기 억제 방식. 선택지 수에 따라 감점 자동 조절 |

- localStorage `xnq_global_settings` 필드: `multipleAnswersScoringMode`, `penaltyMethod`
- 기존에 문항/퀴즈 단위로 관리하던 채점 설정을 전역 단일 정책으로 통합

## 3. 지각 제출 감점 방식 [상태: 결정됨 / XN 범위 외]

Canvas Gradebook에 원점수만 넘기는 구조로 확정됨에 따라, 감점 계산은 Gradebook(Canvas Late Policies)에서 처리. XN Quizzes는 지각 제출 허용 여부(`allowLateSubmit`) 및 허용 기간(`lateSubmitHours`)만 관리.

- Canvas 감점 설정 경로: Gradebook → 우측 상단 톱니바퀴 → Late Policies 탭

---

## 4. 퀴즈-문제은행 연동 방식 [상태: 결정됨: 독립 복사]

퀴즈에 문항 추가 시 문제은행에서 해당 시점의 데이터를 복사해 가져옴. 이후 문제은행 원본을 수정하거나 삭제해도 퀴즈에 반영되지 않음.

---

## 5. 엑셀 문항 업로드 지원 유형 [상태: 결정됨]

행 단위 구조로 표현 가능한 유형만 지원.

| **유형** | **코드** | **지원 여부** |
|---|---|---|
| 객관식 | multiple_choice | 지원 |
| 참/거짓 | true_false | 지원 |
| 단답형 | short_answer | 지원 |
| 서술형 | essay | 지원 (문항 텍스트만) |
| 수치형 이하 나머지 | — | 미지원 |

## 6. 엑셀 일괄 채점 업로드 정책 [상태: 결정됨]

| **항목** | **결정 내용** |
|---|---|
| 학번 미매칭 처리 | 1건이라도 미매칭 시 전체 업로드 취소 |
| 점수 미입력 행 | 건너뜀 허용 (보류 상태로 간주) |
| 배점 초과 | 1건이라도 배점 초과 시 전체 업로드 취소 |
| 오류 발생 시 | 부분 저장 없음. 전체 취소 후 재업로드 유도 |

## 7. 응시 중 문항 수정 범위 [상태: 결정됨]

기존 퀴즈와 동일하게, 이미 제출한 학생이 있더라도 **문항 수정(텍스트, 정답, 선택지 등)을 허용**한다.

- 문항이 변경될 경우, 변경 시점(`questionsModifiedAt`) 기록
- 채점 대시보드에서 "수정 전 제출" / "수정 후 제출" 필터를 제공하여, 문항 변경 이전에 제출한 학생만 구분하여 확인 가능
- **필터에 해당 학생 수 배지 표시**: 수정 전/후 각각 몇 명인지 배지로 표시하여 교수자가 영향 범위를 즉시 파악 가능
- **일괄 재채점 버튼**: "수정 전 제출" 필터 활성화 시, 해당 학생 대상 자동채점 가능 문항 전체를 일괄 재채점하는 버튼 제공
- 교수자는 해당 학생들만 필터링한 뒤 일괄 재채점 또는 수동 점수 조정 가능
- Canvas Classic Quizzes 방식 준용 (수정은 자유, 기존 제출자에 대해서는 교수자가 재채점 트리거)

---

## 8. 영문 대소문자 구분 정답 판정 [상태: 결정됨]

단답형, 수치형 등 자동채점 문항에서 영문 대소문자 구분 여부를 **전역 설정(조직 단위)**으로 제공.

- **퀴즈 관리 페이지 상단 톱니바퀴 아이콘 → "퀴즈 전역 설정" 다이얼로그**에서 ON/OFF
- 기본값: OFF (대소문자 무시 — "Answer"와 "answer"를 동일 정답으로 처리)
- ON 시: "Answer"와 "answer"를 다른 답으로 판정
- 모든 퀴즈에 일괄 적용
- localStorage `xnq_global_settings.caseSensitive` 필드로 관리
