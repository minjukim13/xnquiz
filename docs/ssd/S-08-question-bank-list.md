# S-08. 문제은행 목록

> **라벨 규칙**: `[A]` Canvas 기존 기능 / `[B-#NN]` 학교 요구사항 신규 / `[C]` 자체 도출 개선

## 1. 화면 정보

| **항목** | **내용** |
|---|---|
| 화면 ID | S-08 |
| 라우트 | `/question-banks` |
| 진입 권한 | 교수자 (학생 접근 시 자동 리다이렉트) |
| 진입 경로 | 사이드바 "문제모음" / LearningX LTI 진입 시 section=banks 매핑 |

## 2. 화면 목적

- 교수자가 본인 과목의 모든 문제은행을 한 화면에서 조회/생성/복사/내보내기/가져오기
- 카드 단위로 은행명, 난이도, 문항 수, 최종 수정일을 즉시 식별

## 3. 영역 구성 (Region Map)

| **영역** | **설명** | **주요 컴포넌트** |
|---|---|---|
| 헤더 | "문제모음" 타이틀 + 가져오기/내보내기/새 문제모음 버튼 | h1, Button |
| 본문 | 카드 그리드 (1/2/3열 반응형). 카드 = 은행명(인라인 수정) + 난이도 배지 + 문항 수 + 최종 수정일 | Card |
| 빈 상태 | 은행 0건일 때 안내 + 신규 생성 유도 | EmptyState (BookOpen) |
| 모달 | 새 은행 생성 / 가져오기 / 내보내기 / 삭제 확인 | AddBankModal, ImportBankModal, ExportBankModal, ConfirmDialog |

## 4. 기능 분류

### 4-1. [A] Canvas 기존 기능 - 기존 Canvas 제공 기능 재구현

| **ID** | **기능** | **동작** |
|---|---|---|
| A-08-1 | 문제은행 목록 조회 | 본인 과목의 은행 카드 그리드 표시 |
| A-08-2 | 새 문제은행 생성 | "새 문제모음" → 모달 → 은행명/난이도 입력 → 생성 |
| A-08-3 | 문제은행 삭제 | 카드 휴지통 → ConfirmDialog → 삭제 |
| A-08-4 | 문제은행 진입 | 카드 클릭 → 상세 (S-09) |

### 4-2. [B] 학교 요구사항 반영 - Canvas 에 없거나 부족하여 학교가 요구한 신규 기능

| **ID** | **라벨** | **기능** | **동작** |
|---|---|---|---|
| F-08-1 | [B-#01] | 그룹 단위 난이도 메타 표시 | 카드에 난이도 배지 (쉬움/보통/어려움) |
| F-08-2 | [B-#01] | 최종 수정일 표시 | 카드에 "n일 전 수정" 라벨 |
| F-08-3 | [B-#02] | 문제은행 그룹 복사 | 카드 메뉴 → "복사" → 새 은행 + 모든 문항 복사 |
| F-08-4 | [B-#04] | 다른 과목에서 가져오기 | "가져오기" → 과목 선택 → 은행 다중 선택 → 일괄 import |
| F-08-5 | [B-#04] | 다른 과목으로 내보내기 | "내보내기" → 대상 과목 선택 → 은행 + 문항 export |

### 4-3. [C] 자체 도출 개선 - Canvas 기존 기능도 아니고 학교 요구도 아닌, 프로젝트 자체 발굴 개선

| **ID** | **라벨** | **기능** | **동작** |
|---|---|---|---|
| F-08-6 | [C] | 은행명 인라인 수정 | Pencil 아이콘 → 카드 안에서 직접 수정 (Enter/Escape/Blur) |
| F-08-7 | [C] | 복사 후 즉시 진입 토스트 | 복사 완료 시 "바로가기" 액션이 포함된 토스트 |
| F-08-8 | [C] | 문항 수 카운트 카드 표시 | 은행별 문항 수를 카드에 즉시 노출 |
| F-08-9 | [C] | 1/2/3열 반응형 카드 그리드 | 모바일 1열 → 태블릿 2열 → 데스크톱 3열 |

## 5. 화면 상태 (State)

| **상태** | **트리거** | **표현** |
|---|---|---|
| 로딩 | 초기 fetch 중 | 스켈레톤 |
| 빈 상태 | 은행 0건 | EmptyState + "새 문제모음 만들기" CTA |
| 정상 | 1건 이상 | 카드 그리드 |
| 인라인 편집 중 | 은행명 클릭 | 입력 필드로 변경, Enter 저장 / Escape 취소 |
| 복사 진행 | 복사 트리거 | 토스트 + 즉시 반영 |
| 모달 (생성/가져오기/내보내기/삭제) | 버튼/메뉴 트리거 | 각 모달 활성 |

## 6. 주요 인터랙션

| **#** | **트리거** | **동작** | **결과** |
|---|---|---|---|
| I-1 | 카드 클릭 | `/question-banks/:bankId` | 상세 (S-09) |
| I-2 | "새 문제모음" 버튼 | AddBankModal | 신규 은행 생성 |
| I-3 | "가져오기" | ImportBankModal | 다른 과목 → 본 과목 import |
| I-4 | "내보내기" | ExportBankModal | 본 과목 → 다른 과목 export |
| I-5 | 카드 메뉴 → 복사 | 즉시 복사 + 토스트 | 새 은행 추가 |
| I-6 | 은행명 Pencil 클릭 | 인라인 편집 활성 | Enter/Blur 저장 |
| I-7 | 카드 메뉴 → 삭제 | ConfirmDialog | 확인 시 삭제 |

## 7. 예외/엣지 케이스

- 학생 접근: 즉시 `/` 리다이렉트
- 빈 상태: EmptyState + 새 문제모음 만들기 CTA
- 복사 중 오류: 토스트 안내
- 가져오기 시 동일 은행명: 자동 suffix ("(복사본)") 추가
- 은행명 빈 값으로 저장 시도: 검증 차단
