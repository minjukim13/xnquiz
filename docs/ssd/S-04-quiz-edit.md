## 5. 기능 분류

### 5-1. [A] Canvas 기존 기능 - 기존 Canvas Classic Quizzes 제공 기능 재구현

| **ID** | **기능** | **동작** |
|---|---|---|
| A-04-1 | 기존 퀴즈 데이터 로드 | useParams 의 id 로 getQuiz / getQuizQuestions → 폼 hydrate |
| A-04-2 | 메타/정책/문항 수정 | S-02 와 동일한 폼 사용 |
| A-04-3 | 문항 추가/수정/삭제 | 기존 문항 클릭으로 수정 모달 진입 (S-02 는 신규만) |
| A-04-4 | 게시 상태별 공개여부 토글 | draft 일 땐 토글 비활성, open/closed 일 땐 활성 |
| A-04-5 | 임시저장 (draft 만) | status=draft 인 경우만 임시저장 버튼 노출 |
| A-04-6 | 저장 → 변경 적용 | updateQuiz + setQuizQuestions (전체 재생성 방식) |

### 5-2. [B] 학교 요구사항 반영 - Canvas 에 없거나 부족하여 학교가 요구한 신규 기능

| **ID** | **라벨** | **기능** | **동작** |
|---|---|---|---|
| F-04-1 | [B-#11] | 조건부 재응시 설정 진입 | 응시 정책 영역에서 조건부 재응시 모달 진입. 미응시자/점수 미달자 자동 분류 후 재응시 허용 정책 설정 |
| F-04-2 | [B-#10] | 부분 점수 정책 변경 시 재채점 | 정답/배점 변경 시 기존 응시자에 대한 재채점 옵션 (이전 정답 유지/새 정답만/전원 만점) 선택 |

## 8. 비정상 상태 UX

| **상태** | **트리거** | **표현** |
|---|---|---|
| 로딩 | 초기 fetch 중 | Skeleton (탭 헤더 + 섹션) |
| draft 편집 | quiz.status=draft | "임시저장" + "저장하기" 모두 활성, 공개여부 토글 비활성 |
| open 편집 | quiz.status=open | "저장하기" 만, 공개여부 토글 활성, 재채점 옵션 안내 |
| closed 편집 | quiz.status=closed | "저장하기" + 마감일을 미래로 변경하거나 비우면 재오픈 안내 |
| dirty | hasUnsavedChanges=true | beforeunload 경고 활성, 취소 시 ConfirmDialog |
| 재채점 트리거 | regradeMap 비어있지 않음 | 저장 직전 "재채점 옵션 적용" 안내 |
| 저장 진행 | API 호출 중 | 버튼 비활성 + 스피너 |
| 빈 상태 | 해당 없음 (편집 대상 퀴즈는 항상 존재) | 해당 없음. fetch 실패는 에러 상태로 처리 |
| 에러 | 퀴즈 fetch 실패 / 저장 실패 | ErrorState (재시도) / 토스트 + 폼 데이터 유지 |
| 권한 없음 | 학생이 URL 직접 진입 | 즉시 `/` 로 리다이렉트 |
| 오프라인 | 네트워크 단절 | 전역 OfflineBanner, 저장 버튼 비활성 |

## 9. 예외 / 엣지 케이스

- 게시(open) 후 문항 삭제: 기존 응시자 답안에서 해당 문항 점수가 빠지므로 재채점 자동 트리거
- 응시 중인 학생 있음 + 문항 수정: 수정 후 다음 응시자부터 적용 (이미 제출한 응시자에는 재채점 옵션으로 적용)
- 시작일/마감일/이용종료일은 모두 비워둘 수 있음. 단, 값이 있는 경우 시작일 ≤ 마감일 ≤ 이용종료일 순서를 위반하면 검증 차단
- 자동 제출 비활성화 ON 인데 이용종료일 미설정 → 무한정 응시 방지를 위해 검증 차단
- closed 상태에서 마감일을 미래로 변경하거나 비우면 자동 재오픈 (status=open). 단, 지각 제출 ON + lateSubmitDeadline 이 과거이면 closed 유지
- 임시저장 시도 시 status 가 draft 가 아니면 "임시저장은 초안 상태에서만 가능" 알림
- 폼이 dirty 인데 새로고침 시도 → 브라우저 native 경고 + 앱 내 ConfirmDialog
- 전체 문항 삭제 후 저장 → S-02 와 동일하게 검증 차단 (status 가 draft 아닌 경우)
