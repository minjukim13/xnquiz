## 8. 비정상 상태 UX

가이드 §5 비정상 상태 UX 항목 전체 정의. 해당 없는 상태는 "해당 없음" 명시.

| **상태** | **트리거** | **표현** |
|---|---|---|
| 로딩 | 데이터 fetch 중 | Skeleton {N}개 placeholder |
| 빈 상태 | 데이터 0건 | EmptyState (안내 문구 + 보조 텍스트) |
| 에러 | API 실패 | ErrorState (재시도 버튼) |
| 권한 없음 | 본 화면에 권한 없는 역할이 직접 URL 진입 | AccessDenied (목록으로 이동 링크) / 해당 없음 |
| 오프라인 | 네트워크 단절 | 전역 OfflineBanner |
