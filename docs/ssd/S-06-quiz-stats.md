
## 5. 기능 분류

### 5-1. [A] Canvas 기존 기능 - 기존 Canvas Classic Quizzes 제공 기능 재구현

| **ID** | **기능** | **동작** |
|---|---|---|
| A-06-1 | 평균/최고/최저/표준편차 표시 | 채점 완료 학생의 점수 통계 산출 |
| A-06-2 | 점수 분포 막대 차트 | 점수 구간별 인원 수 표시 |
| A-06-3 | 문항별 정답률 표시 | 문항당 정답 학생 비율 |
| A-06-4 | 학생 목록 + 정렬/검색 | 이름/학번/학과/소요시간/제출일시/점수 정렬 + 검색 |
| A-06-5 | 응시율/제출율 표시 | 전체 학생 대비 응시/제출/채점 비율 |

### 5-2. [B] 학교 요구사항 반영 - Canvas 에 없거나 부족하여 학교가 요구한 신규 기능

| **ID** | **라벨** | **기능** | **동작** |
|---|---|---|---|
| F-06-1 | [B-#13] | 학생별 성적 xlsx 다운로드 | 학생명/학번/학과/제출일시/점수/자동채점/수동채점/채점상태 컬럼 |
| F-06-2 | [B-#13] | 문항분석 xlsx 다운로드 | 2시트 (문항별 통계 + 퀴즈 요약) |
| F-06-3 | [B-#09] | 출력 가능한 형식 제공 | xlsx 외에도 화면 캡처 가능한 정렬된 표 형식 |

### 5-3. [C] 자체 도출 개선 - Canvas 기존 기능도 아니고 학교 요구도 아닌, 프로젝트 자체 발굴 개선

| **ID** | **라벨** | **기능** | **동작** |
|---|---|---|---|
| F-06-4 | [C] | 평균선 기준선 표시 | 점수 분포 차트에 평균을 점선으로 시각화하고, 평균이 속한 구간 막대를 강조 색상으로 표시 |
| F-06-5 | [C] | 점수 자동 구간 분할 | 퀴즈 총점에 따라 10점/5점/1점 단위로 구간 폭 자동 결정 |
| F-06-6 | [C] | 응시 현황 5단 progress bar | 수강 인원/응시 완료/미제출/채점 완료/채점 대기를 비율 막대 + 인원 동시 표기 |
| F-06-7 | [C] | 상위/중위/하위 27% 자동 분할 | 채점 완료 학생 점수에서 상위 27% / 하위 27% 경계 점수를 자동 산출하여 표시 |
| F-06-8 | [C] | 문항별 득점률 색상 코딩 | 70% 이상 녹색(쉬움) / 40~69% 주황(보통) / 40% 미만 빨강(어려움) / 채점 전 회색 |
| F-06-9 | [C] | 70%/40% 기준선 | 수평 막대그래프에 점선 기준선으로 난이도 경계 시각화 |
| F-06-10 | [C] | 자동 난이도 배지 | 위 색상 코딩과 동일 기준으로 표 행에도 배지 부여 |
| F-06-11 | [C] | 문항별 상세 표 - 인라인 progress bar + 수치 동시 | 한 셀 안에 막대 + 숫자 동시 표시 (Canvas 는 분리) |
| F-06-12 | [C] | 변별도(D) + 이분상관(r_pb) + Cronbach α | 문항분석 xlsx 시트에 심화 지표 자동 계산 |
| F-06-13 | [C] | 학생별 성적 화면 → 답안 확인 진입 | "답안 확인" 버튼으로 채점 대시보드(S-05) 의 해당 학생 모드로 직접 진입 |
| F-06-14 | [C] | 채점 미완료 표시 | gradedCount < totalCount 인 문항은 "진행중(n/m명)" 라벨 |

## 8. 비정상 상태 UX

| **상태** | **트리거** | **표현** |
|---|---|---|
| 로딩 | 초기 fetch 중 | Skeleton (요약 카드 + 차트) |
| 빈 상태 (응시자 0) | 응시 학생 없음 | "채점 완료된 학생이 없습니다" 안내 |
| 빈 상태 (채점 미완료) | 채점 완료자 0 | 점수 분포 차트 자리에 안내 |
| 정상 | 채점 완료 1명 이상 | 차트 + 표 정상 렌더 |
| 표준편차 0 (단일/동일 점수) | 학생 1명 또는 동일 점수 | 표준편차 0 표시 (오류 아님) |
| 무효 시간 | 시작/종료 시간 invalid | 소요시간 "-" 표시 |
| 에러 | 통계 산출 API 실패 | ErrorState + 재시도 |
| 권한 없음 | 학생이 URL 직접 진입 | 즉시 `/` 로 리다이렉트 |
| 오프라인 | 네트워크 단절 | 전역 OfflineBanner, xlsx 다운로드 비활성 |

## 9. 예외 / 엣지 케이스

- 미응시자(startTime null): 마감 후 자동 0점 처리되며, 평균/분포 통계 산출 시 전체 수강생 기준에 포함 (S-05 채점 대시보드 진행률 분모 정의와 동일)
- 채점 미완료: 평균/표준편차 계산에서 제외 (채점 완료자만)
- 27% 구간 산출 실패 (학생 < 4명 등): "-" 표시
- 분산 0 (1명 또는 동일 점수): stdev = 0 으로 표기
- xlsx 다운로드 시 한글 컬럼명 + Cronbach α/D/r_pb 등 심화 지표 포함 (화면엔 비표시)
- 문항이 0개인 퀴즈 (이론적): 문항별 표는 빈 상태
