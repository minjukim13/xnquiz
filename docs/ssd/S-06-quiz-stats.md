## 7. 예외/엣지 케이스

- 미응시자(startTime null): 자동 0점 처리되었어도 통계의 평균/분포에는 포함 (분모 일치 정책)
- 채점 미완료: 평균/표준편차 계산에서 제외 (채점 완료자만)
- 27% 구간 산출 실패 (학생 < 4명 등): "-" 표시
- 분산 0 (1명 또는 동일 점수): stdev = 0 으로 표기
- xlsx 다운로드 시 한글 컬럼명 + Cronbach α/D/r_pb 등 심화 지표 포함 (화면엔 비표시)
- 문항이 0개인 퀴즈 (이론적): 문항별 표는 빈 상태
