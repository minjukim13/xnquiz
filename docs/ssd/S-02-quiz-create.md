## 4. 기능 분류

### 4-1. [A] Canvas 기존 기능 - 기존 Canvas 제공 기능 재구현

| **ID** | **기능** | **동작** |
|---|---|---|
| A-02-1 | 제목 / 설명 입력 | 필수값(제목), 설명은 RichText 지원 |
| A-02-2 | 퀴즈 유형 선택 | 평가용 / 연습용 |
| A-02-3 | 응시 기간 설정 | 시작일시 / 마감일시 / 이용 종료일시 |
| A-02-4 | 시간 제한 | 토글 + 분 수 + 자동제출 비활성화 옵션 |
| A-02-5 | 재응시 횟수 + 점수 정책 | 최대 횟수 (2~10/무제한) + 점수 반영 정책 (최고/최신/평균) |
| A-02-6 | 선택지 무작위 / 문항 순서 무작위 | Shuffle 옵션 |
| A-02-7 | 한 문항씩 표시 + 응답 후 잠금 | One question at a time + 잠금 |
| A-02-8 | 학생별 추가 기간 (AssignmentOverrides) | 특정 학생/그룹에게 별도 마감일 부여 |
| A-02-9 | 액세스 코드 / IP 제한 | 응시 진입 시 사전 검증 |
| A-02-10 | 문항 직접 작성 | 12개 유형 모달로 작성 |
| A-02-11 | 문항 드래그 정렬 | GripVertical 핸들로 순서 변경 |
| A-02-12 | 안내사항 입력 | 응시 전 학생에게 노출되는 안내 텍스트 |
| A-02-13 | 학생 공개여부 토글 | 저장 시점에 학생에게 노출할지 |
| A-02-14 | 임시저장 / 저장 분리 | 임시저장은 제목만 필수(학생 비공개 상태로 저장), 저장은 전체 검증 |
| A-02-15 | 주차 / 차시 메타 + LTI 자동 프리필 | LMS 과목에 등록된 주차/차시 옵션을 불러와 선택 (과목별 가변, 예: 어떤 과목은 3주차까지만). 기본값 1주차 1차시. LearningX 진입 시 localStorage 의 week/session 자동 채움 |

### 4-3. [C] 자체 도출 개선 - Canvas 기존 기능도 아니고 학교 요구도 아닌, 프로젝트 자체 발굴 개선

| **ID** | **라벨** | **기능** | **동작** |

| F-02-11 | [C] | 단계 모달 푸터 통일 (이전 좌측 / 취소+다음 우측) | 모든 스텝 모달이 동일 푸터 레이아웃 |

| F-02-15 | [C] | 재응시 + 점수 미공개 경고 | 재응시 ON 인데 점수 공개 시점이 모호할 때 저장 직전 확인 다이얼로그 |

## 7. 예외/엣지 케이스

- 제목 누락 → "제목을 입력해 주세요" AlertDialog
- 시작일/마감일/이용종료일은 모두 비워둘 수 있음. 단, 값이 있는 경우 시작일 ≤ 마감일 ≤ 이용종료일 순서를 위반하면 날짜 모순 안내
- 지각 제출 ON 인데 마감일 미설정 → 지각 마감 무의미, 검증 차단
- 자동 제출 비활성화 ON 인데 이용종료일 미설정 → 무한정 응시 방지를 위해 검증 차단
- 시간 제한 ON 인데 분 수 0/음수 → 검증 차단
- 문항 0개 + 저장(draft 외) → 검증 차단
- 학생 어사인먼트 중복 (동일 학생 ID 다중 등록) → hasDuplicateStudent 검출 후 검증 차단
- 재응시 1회 → 점수 정책 자동 null 처리 (의미 없음)
- 한 문항씩 표시 ON 시 응답 후 잠금 자동 강제
- LTI 모드: localStorage 의 week/session 값이 LMS 과목에 등록되지 않은 항목이면 무시 후 기본값(1주차/1차시) 적용
- 임시저장 시에는 마감일/문항/접근제어 검증 모두 skip
