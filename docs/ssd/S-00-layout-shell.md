## 2. 화면 목적

- 전 화면 공통 사이드바 + 메인 영역 셸

- LTI 모드에선 사이드바 숨기고 콘텐츠만 풀너비로 표시 (LearningX 호스트 환경 통합)

## 3. 영역 구성 (Region Map)

| **영역** | **설명** | **주요 컴포넌트** |
|---|---|---|

| Suspense 폴백 | 라우트 전환 중 lazy chunk 로딩 시 스피너 | ContentFallback |

## 4. 기능 분류

### 4-1. [A] Canvas 기존 기능 - 기존 Canvas 제공 기능 재구현

| **ID** | **기능** | **동작** | **구현 위치** |
|---|---|---|---|
| A-00-1 | 사이드바 네비게이션 | "퀴즈" / "문제모음" 메뉴 + 활성 라우트 시각 표시 | NavLinks |
| A-00-2 | 메인 영역 라우팅 | react-router Outlet 으로 자식 라우트 렌더 | Outlet |
| A-00-3 | 역할별 메뉴 분기 | 교수자/학생에게 보이는 메뉴 차이 (학생은 "퀴즈" 만) | navItems 분기 |
| A-00-4 | 모바일 반응형 네비 | 모바일은 햄버거 → Sheet 모달 네비 | Sheet |

### 4-2. [B] 학교 요구사항 반영 - Canvas 에 없거나 부족하여 학교가 요구한 신규 기능

| **ID** | **라벨** | **기능** | **동작** | **구현 위치** |
|---|---|---|---|---|
| (해당 없음) | - | 학교 요구사항으로 정의된 레이아웃 기능 없음 | - | - |

### 4-3. [C] 자체 도출 개선 - Canvas 기존 기능도 아니고 학교 요구도 아닌, 프로젝트 자체 발굴 개선

| **ID** | **라벨** | **기능** | **동작** | **구현 위치** |
|---|---|---|---|---|
| F-00-1 | [C] | LTI 모드 자동 감지 + 셸 숨김 | localStorage 의 `xnq_lti_active==='1'` 일 때 사이드바/모바일 헤더 숨기고 콘텐츠 풀너비 | LTI 모드 분기 |
| F-00-2 | [C] | 라우트 프리페치 | NavLinks 의 hover/focus/touchStart 이벤트로 lazy chunk 미리 로드 | prefetchRoute, PREFETCH_MAP |
| F-00-3 | [C] | 라우트 전환 시 사이드바 유지 | Layout Route 패턴으로 사이드바를 부모에 두고 Outlet 만 교체 | Layout Route + Outlet |
| F-00-4 | [C] | Suspense 폴백 (스피너) | 라우트 전환 중 lazy 로드되는 동안 main 영역에만 스피너 (사이드바는 유지) | Suspense + ContentFallback |
| F-00-5 | [C] | 학생 모드 정보 배너 | 학생 역할일 때 본인 이름/학번을 메인 영역 상단에 배너로 표시 | 학생 배너 |
| F-00-6 | [C] | 모바일 시트 자동 닫힘 | 모바일에서 네비 클릭 시 Sheet 자동 닫힘 | setMobileNavOpen(false) |
| F-00-7 | [C] | requestIdleCallback 폴백 | 프리페치 시 브라우저 idle 시간 활용, 미지원 시 setTimeout 300ms 폴백 | prefetch util |

## 5. 화면 상태 (State)

| **상태** | **트리거** | **표현** |
|---|---|---|
| 일반 (데스크톱) | 768px 이상 | 사이드바 + 메인 |

| LTI 모드 | localStorage `xnq_lti_active=1` | 사이드바/헤더 숨김, 콘텐츠 풀너비 |

## 7. 예외/엣지 케이스

- requestIdleCallback 미지원 브라우저: setTimeout 300ms 폴백
- 라우트 전환 중 사용자가 다른 라우트 클릭: 마지막 클릭만 적용
- LTI 진입 시 토큰 누락: 일반 모드로 fallback
- LTI 모드에서 사용자가 직접 사이드바 메뉴 호출 시도: 메뉴 자체가 없으므로 차단됨
- 학생 모드 배너 - currentStudent 정보 미존재 시 배너 숨김
