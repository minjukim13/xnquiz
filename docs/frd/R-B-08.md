# R-B-08. 시험 출제 기능 직관성 개선

## 1. 출처 / 컨텍스트

- **카테고리**: B (고객사 요구 #08)
- **영향 사용자**: 교수자, 조교, 시험 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 탭명("세부 사항 / 문제 작성")이 시스템 내부 분류 중심이라 사용자 작업 의도와 맞지 않음
- **P2.** 시험 세팅 단계와 문제 출제 단계가 한 흐름으로 명확히 구분되지 않음
- **P3.** 단계별 옵션 설명/가이드가 부족해 기능 의미를 사용자가 추정해야 함

## 3. 개선 가설
탭명을 행동 유도형으로 변경하고 단계 진행 버튼을 명시하면, 사용자는 화면 해석 시간 없이 자신의 현재 위치와 다음 작업을 즉시 인지한다.
