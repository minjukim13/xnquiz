# R-B-12. 시험 목록 화면 제공

## 1. 출처 / 컨텍스트
- **PCD**: [5017829388](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5017829388) 제출물·목록 관리 기능 / 시험 목록 화면 제공 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #12)
- **영향 사용자**: 교수자, 시험 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 시험과 주차/차시의 매칭 관계를 한눈에 확인하기 어려움
- **P2.** 기존 Canvas "시험 및 설문" 메뉴는 주차 정보 노출 불가, 주차학습은 시험만 모아 보기 불가

## 3. 개선 가설
주차/차시 정보가 함께 노출되는 시험 전용 목록 화면(xn 퀴즈)을 제공하면, 운영자는 화면 이동 없이 주차별 시험 배치 현황을 한 화면에서 확인할 수 있다.
