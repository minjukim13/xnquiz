# R-B-13. 제출물 다운로드

## 1. 출처 / 컨텍스트
- **PCD**: [5016715342](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5016715342) 제출물·목록 관리 기능 / 제출물 다운로드 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #13)
- **영향 사용자**: 교수자, 조교, 시험 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 파일 업로드형 시험에서 문항 단위 제출물 식별 불가 (전체 섞여 다운로드)
- **P2.** 퀴즈 통계 내 문항별 다운로드 버튼이 실제로는 전체 문항 대상으로 동작 (기대 불일치)
- **P3.** 시험 단위/문항 단위 관리 기준 불명확

## 3. 개선 가설
다운로드 결과를 문항 기준으로 구조화하고 버튼 위치와 동작 범위를 일치시키면, 운영자는 별도 분류 작업 없이 채점/검토 목적에 맞는 제출물을 바로 확보할 수 있다.
