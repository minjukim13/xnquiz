# R-C-07. 랜덤 출제 고도화

## 1. 출처 / 컨텍스트
- **PCD**: [5005246509](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5005246509) 7. 시험 설계 및 랜덤 출제 기능 / 랜덤 출제 고도화 (작성자: 김범수)
- **카테고리**: C (고객사 요구 #07)
- **영향 사용자**: 교수자, 조교, 시험 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 랜덤 출제가 "정책 설계" 가 아닌 "반복 선택 작업" 중심으로 동작
- **P2.** 출제 기준을 그룹 단위로 구조화해 설계하기 어려움
- **P3.** 난이도 기반 랜덤 출제 확장의 선행 기준(메타데이터) 부족

## 3. 개선 가설
랜덤 출제를 "1단계 다중 그룹 선택 + 2단계 그룹별 옵션 설정" 으로 분리 + 미리보기를 제공하면, 사용자는 시험 설계 의도를 정책 단위로 표현할 수 있고 출제 전 검토가 가능해진다.

## 6. 통과 기준 (Acceptance Criteria)

| **#** | **기준** | **검증 방식** |
|---|---|---|
| AC-1 | 다중 그룹 선택 → 그룹별 옵션 설정의 2단계 흐름이 작동한다 | 코드/캡처 |
