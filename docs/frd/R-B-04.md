# R-B-04. 문항 등록 효율화

## 1. 출처 / 컨텍스트
- **PCD**: [5004656658](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5004656658) 4. 문제은행 및 문항 자산 관리 / 문항 등록 효율화 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #04)
- **영향 사용자**: 교수자, 조교

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 대량 문항 등록을 전제로 한 입력 경로 부족 (단건 입력 중심)
- **P2.** 동일 구조 반복 입력으로 인한 작업 피로/오류 가능성 증가
- **P3.** 무들 등 외부 시스템의 일괄 업로드 경험 대비 격차

## 3. 개선 가설
범용 유형(객관식/OX/단답) 대상 엑셀 일괄 업로드 경로를 제공하고, 업로드 전 검증을 통해 오류를 미리 노출하면, 사용자는 반복 입력 부담 없이 문제은행을 빠르게 적재할 수 있다.
