# R-C-10. 부분 점수 정책 개선

## 1. 출처 / 컨텍스트
- **PCD**: [5013635098](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5013635098) 10. 채점·성적·재응시 기능 / 부분 점수 정책 개선 (작성자: 김범수)
- **카테고리**: C (고객사 요구 #10)
- **영향 사용자**: 교수자 (출제/채점), 학생

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 부분 점수 정책이 시스템 기본으로 고정되어 운영 의도 반영 불가
- **P2.** 현재 산정 방식(정답 가점/오답 동일 단위 차감)이 국내 평가 관행과 직관 불일치

## 3. 개선 가설
부분 점수 적용 여부와 산정 방식을 정책 옵션으로 분리해 출제자가 선택할 수 있게 하면, 평가 의도에 맞는 채점 정책을 설계할 수 있고 학생은 결과를 예측 가능하게 이해한다.
