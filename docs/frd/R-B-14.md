# R-B-14. 답변 입력 설정창 확대

## 1. 출처 / 컨텍스트
- **PCD**: [5017567266](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5017567266) 제출물·목록 관리 기능 / 답변 입력 설정창 확대 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #14)
- **영향 사용자**: 교수자, 조교

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 답변 입력 영역이 좁아 작성/확인 효율 낮음 (객관식, 빈칸, 복수 빈칸, 목록 선택 유형)

## 3. 개선 가설
답변 설정창의 입력 영역을 가로/세로 모두 확장하면, 출제자는 긴 답변이나 다항 선택지를 스크롤/재확인 없이 한눈에 작성·검토할 수 있다.
