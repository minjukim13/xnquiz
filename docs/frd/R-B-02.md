# R-B-02. 문제은행 그룹 복사 개선

## 1. 출처 / 컨텍스트
- **PCD**: [5002657805](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5002657805) 2. 문제은행 및 문항 자산 관리 / 문제은행 그룹 복사 개선 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #02)
- **영향 사용자**: 교수자, 조교, 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 문제은행 그룹 재사용 경로가 직관적이지 않음 (Canvas 복사 기능 의존)
- **P2.** 즐겨찾기 기능이 학기 종료 후 노출 제약으로 장기 재사용 수단 한계

## 3. 개선 가설
문제은행 그룹 단위 전용 내보내기/가져오기 진입점을 제공하면, 사용자는 Canvas 복사 절차 학습 없이 목적어("그룹") 기준으로 재사용을 즉시 수행할 수 있다.

## 5. Before / After 비교

| **측정 항목** | **Before (Canvas)** | **After (xnquiz)** | **개선 효과** |
|---|---|---|---|
| 진입점 위치 | Canvas 과목 복사 메뉴 | 문제은행 화면 내 직접 진입 | 목적어 기준 직관 진입 |
| 절차 단계 수 | 다중 단계 (과목 복사 → 항목 선택 → 실행) | 모달 1개 단계 | 단계 단순화 |
