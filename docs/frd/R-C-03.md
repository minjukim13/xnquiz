# R-C-03. 시험 문제 복사 개선

## 1. 출처 / 컨텍스트
- **PCD**: [5002428457](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5002428457) 3. 문제은행 및 문항 자산 관리 / 시험 문제 복사 개선 방안 (작성자: 김범수)
- **카테고리**: C (고객사 요구 #03)
- **영향 사용자**: 교수자, 조교, 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 개별 문항 재사용이 문제은행 매개를 거쳐야 하는 우회 구조
- **P2.** 시험 전체 복사가 Canvas 복사 기능에 의존, 진입점이 시험 기능 외부

## 3. 개선 가설
시험 기능 안에서 "문항"과 "시험"을 모두 직접 복사할 수 있도록 진입점을 제공하면, 사용자는 문제은행/Canvas 구조 학습 없이 목적어 단위로 재사용을 수행한다.
