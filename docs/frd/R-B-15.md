# R-B-15. 답변 입력창 줄바꿈

## 1. 출처 / 컨텍스트
- **PCD**: [5017698320](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5017698320) 제출물·목록 관리 기능 / 답변 입력창 줄바꿈 기능 요청 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #15)
- **영향 사용자**: 교수자, 조교

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 답변 입력값을 한 줄로만 관리해 가독성 낮음 (긴 정답/문장형)
- **P2.** 복수 정답 또는 대체 답변의 시각적 구분 기준 불명확
- **P3.** 문항 유형별 답변 작성 방식 유연성 부족

## 3. 개선 가설
Shift+Enter / Ctrl+Enter 줄바꿈 입력을 답변 설정창 전반에 일관 적용하면, 출제자는 긴 정답이나 복수 정답을 줄 단위로 구분해 작성·검수할 수 있다.
