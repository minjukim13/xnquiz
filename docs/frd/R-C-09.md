# R-C-09. 출력 및 다운로드 기능 개선

## 1. 출처 / 컨텍스트
- **PCD**: [5004722209](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5004722209) 9. 출력 및 문서화 기능 / 출력 및 다운로드 기능 개선 (작성자: 김범수)
- **카테고리**: C (고객사 요구 #09)
- **영향 사용자**: 교수자, 조교, 시험 운영자

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 온라인 시험 문제를 외부 문서 형태로 확보하기 어려움
- **P2.** 학생 답안지를 개별/일괄 단위로 문서화하기 어려움
- **P3.** 출력물 포함 정보 정책 기준 불명확

## 3. 개선 가설
시험 문제 / 개별 답안 / 다수 답안 일괄을 PDF 단위로 명시 분리해 다운로드 진입점을 제공하면, 운영자는 화면 캡처/우회 없이 행정 목적별로 산출물을 즉시 확보할 수 있다.
