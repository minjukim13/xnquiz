# R-B-06. 문항 제목 설정

## 1. 출처 / 컨텍스트
- **PCD**: [5005148199](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5005148199) 6. 문제은행 및 문항 자산 관리 / 문항 제목 설정 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #06)
- **영향 사용자**: 교수자, 학생

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 교수자가 설정한 문항 제목이 학생 화면에는 기본값(문제1, 문제2)으로만 표시되어 설정값과 노출값 불일치
- **P2.** 편집 경험과 학생 참여 경험 간 일관성 부족
- **P3.** 문항 제목의 제품 내 역할(내부용/노출용) 모호

## 3. 개선 가설
문항 제목을 "선택형 제목" 으로 재정의하고 placeholder 로 인지시키며, 입력 시에만 학생 화면에 노출하면, 교수자는 설정값 신뢰도가 높아지고 학생은 문항 맥락을 추가로 이해할 수 있다.

## 5. Before / After 비교

| **측정 항목** | **Before (Canvas)** | **After (xnquiz)** | **개선 효과** |
|---|---|---|---|
| 교수자 설정 | 제목 입력 가능하나 노출 안 됨 | 제목 입력 시 학생 화면에 반영 | 설정값 신뢰도 |
| 기본값 처리 | "문제1, 문제2" 강제 | placeholder, 미입력 시만 자동 번호 | 사용자 입력 vs 기본값 분리 |
