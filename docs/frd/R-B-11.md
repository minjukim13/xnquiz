# R-B-11. 조건부 퀴즈 재응시

## 1. 출처 / 컨텍스트
- **PCD**: [5014224922](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5014224922) 채점·성적·재응시 기능 / 조건부 퀴즈 재응시 (작성자: 김범수)
- **카테고리**: B (고객사 요구 #11)
- **영향 사용자**: 교수자, 조교, 학생

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 재응시 대상자 선별이 수동 운영(개별 학생 응시 여부/점수 확인)에 의존
- **P2.** 재응시 운영 기준(미응시자/점수 미달자/혼합)이 기능으로 구조화되지 않음

## 3. 개선 가설
재응시를 "전체 개방" 이 아닌 "조건 기반 자동 분류 + 정책 설정" 흐름으로 제공하면, 교수자는 수동 선별 없이 보완 학습 기회를 목적성 있게 부여할 수 있다.
