# R-C-01. 문항 메타데이터 및 분류 강화

## 1. 출처 / 컨텍스트
- **PCD**: [5000691857](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5000691857) 1. 문제은행 및 문항 자산 관리 / 문항 메타데이터 및 분류 강화 (작성자: 김범수)
- **카테고리**: C (고객사 요구 #01)
- **영향 사용자**: 교수자, 조교

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 문항 자산의 분류 기준이 부족해 선별과 재사용이 비효율적
- **P2.** 문제 단위와 그룹 단위 관리 관점이 분리되지 않아 운영 일관성 떨어짐

## 3. 개선 가설
문항/문제은행 그룹 각각에 난이도 메타데이터를 분리해 등록·표시하면, 교수자는 검색/선별 단계에서 추가 탐색 없이 적합한 자산을 식별할 수 있다.
