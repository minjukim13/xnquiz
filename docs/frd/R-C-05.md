# R-C-05. 정답 판정 옵션

## 1. 출처 / 컨텍스트
- **PCD**: [5004787746](https://xinics.atlassian.net/wiki/spaces/XP2/pages/5004787746) 5. 문제은행 및 문항 자산 관리 / 정답 판정 옵션 (작성자: 김범수)
- **카테고리**: C (고객사 요구 #05)
- **영향 사용자**: 교수자 (출제), 학생 (응시)

## 2. 해결 대상 UX 문제 (PCD 인용)
- **P1.** 입력형 문항의 정답 판정이 표기 차이(대소문자 등)에 과도하게 민감
- **P2.** 문장형 답안의 띄어쓰기 차이가 자동 채점 오류로 체감
- **P3.** 문항 의도별 채점 기준을 출제자가 통제하기 어려움

## 3. 개선 가설
대소문자 구분/띄어쓰기 적용 옵션을 문항 단위로 제공하면, 출제자는 문항 의도에 맞는 채점 기준을 설계할 수 있고 학생은 표기 차이만으로 오답 처리되는 경험이 줄어든다.
