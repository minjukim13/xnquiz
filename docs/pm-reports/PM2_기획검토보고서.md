### Critical/High

| 번호 | 케이스 | 우선순위 | 현재 처리 |
|---|---|---|---|
| E-04 | 지각 제출 (마감 후 제출) | **High** | `allowLateSubmit` 필드만 존재, 검증 로직 없음. 마감 후에도 status가 open이면 응시 가능 |

## 상태 전이 현황

```
draft → open (발행 버튼) ← 구현 완료
open → grading (마감 도래) ← 미구현
open → closed (수동 마감) ← 미구현
grading → closed (전체 채점 완료) ← 미구현
draft → scheduled (예약 발행) ← 미구현
```

**grading 상태 문제**: open과 동일 레이블("진행중")/색상(green)으로 표시됨. 의미 구분 필요.
