## 2. 확정 정책 (변동 없음)

### 재응시 정책
- 이전 응시 제출 완료 후 즉시 재응시 가능
- 무제한(-1) 상한 없음 (실서비스 시 쿨다운 추가 필요)
- 마감 후 시작은 차단, 응시 중 마감 도래 시 해당 건만 유효

### 점수 공개 정책
- 자동채점 점수는 응시 직후 즉시 표시
- 수동채점 포함 최종 점수는 showScore + 공개 기간으로 제어 (MVP2 구현 필요)

### 상태 전이
```
draft → scheduled → open → grading → closed
```
- 현재 draft→open만 구현. 나머지 자동 전이는 MVP2 범위

## 3. 잔존 정책 공백 (해결 필요)

| 항목 | 공백 내용 | 우선순위 |
|---|---|---|

| **수동채점 포함 점수 정책** | 1차 수동채점 완료 / 2차 미완료 시 "최고 점수" 비교 기준 미정의 | **High** |

| **단답형 autoGrade 불일치** | QUIZ_TYPES는 `'partial'`, 실제는 수동채점 처리 | **Medium** |
